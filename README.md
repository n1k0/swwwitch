Swwwitch
========

Swwwitch is a Web version of [Switch](https://play.google.com/store/apps/details?id=com.monkeymoon.switch). It's playable online [here](http://n1k0.github.io/swwwitch).

![](swwwitch.gif)

Hacking on this
---------------

```
$ npm install -g react-tools
$ make watch
```
